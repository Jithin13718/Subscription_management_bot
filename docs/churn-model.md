# Churn Prediction Model

## Objective
Reduce churn by 20% through proactive engagement.

## Inputs
- Usage frequency
- Billing history
- Support tickets
- Cancellation intent signals

## Outputs
- **Risk Score (0–1)** → Probability of churn
- **Retention Strategy** → Discount, pause, feature highlight

## Techniques
- Logistic Regression / Random Forest baseline
- Feature engineering on usage + billing data
- Retraining every 30 days with new subscription data

## Metrics
- **Churn Reduction Target:** 20%
- **Upgrade Conversion Target:** 30%
- **Automated Billing Inquiry Coverage:** 80%
